# 💻 Roblox Build Logic - CPU Web IDE & Compiler 67

โปรเจกต์นี้คือ **Web IDE & Assembler** สำหรับเขียนโค้ดควบคุมระบบสมองกลคอมพิวเตอร์ (CPU) ที่สร้างขึ้นภายในเกม Roblox (แมพ Build Logic) โดยตัวเว็บจะทำหน้าที่แปลงภาษามนุษย์ (Assembly/ASCII) ให้กลายเป็นรหัสเครื่อง **Machine Code (8-Bit)** และเปิดช่องทางให้คอมพิวเตอร์ในเกม Roblox สามารถดึงข้อมูล (HTTP GET) เข้าไปรันในเกมได้โดยตรง

---

## 🛠️ วิธีการติดตั้งและเปิดใช้งานเซิร์ฟเวอร์ (Setup)

1. **เปิด GitHub Codespaces**
   - เข้าไปที่ Repository นี้บน GitHub
   - กดปุ่ม **Code** (สีเขียว) -> เลือกแท็บ **Codespaces** -> กด **Create codespace on main**

2. **สั่งรันเซิร์ฟเวอร์**
   - มองหาช่อง **Terminal (จอดำ)** ด้านล่าง แล้วพิมพ์คำสั่งนี้เพื่อเปิดระบบ:
     ```bash
     node server.js
     ```

3. **ปลดล็อกพอร์ตให้เป็นสาธารณะ (สำคัญมาก! ⚠️)**
   - ไปที่แท็บ **Ports** (อยู่ข้างๆ ช่อง Terminal ด้านล่าง)
   - จะเจอพอร์ตเลข `3000`
   - คลิกขวาที่คำว่า **Private** แล้วเปลี่ยนให้เป็น **Public** เพื่อเปิดทางให้เกม Roblox สามารถวิ่งเข้ามาดึงข้อมูลได้

---

## 🚀 วิธีการใช้งานระบบเขียนโค้ด (Web IDE)

1. คัดลอกลิงก์จากช่อง **Local Address** ในแท็บ Ports (เช่น `https://...-3000.app.github.dev/`) นำไปเปิดในหน้าเว็บเบราว์เซอร์
2. คุณจะพบกับหน้าจอเขียนโค้ดสีดำ-เขียว (Hacker Style)
3. พิมพ์คำสั่งสั่งการ CPU ที่ช่องด้านซ้าย เช่น:
   ```text
   LDA 5   // โหลดเลข 5 ใส่รีจิสเตอร์ A
   ADD 3   // บวกเลข 3 เข้าไป
   OUT     // ส่งผลลัพธ์ออกหน้าจอ
   HLT     // สั่งให้เครื่องหยุดทำงาน
